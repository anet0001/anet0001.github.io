HTML tags I've learnt in class

The heading tag (h1 - h6)
This is used to denote headings in a html document.
The header tag
This tag defines the header section of a webpage. it typically contains introductory content.
The article tag
This is used to define independent, self contained pieces of content on a webpage.
The section tag
This is used to divide the content of a webpage into sections logically.
The blockquote tag
This is commonly used to indicate a block of quoted text on a website.
